Grid Classes

The Bootstrap grid system has 5 breakpoints:

  • Extra small (<576px wide).
    Class prefix: ega-col-
  • Small (>=576px wide)
    Class prefix: ega-col-sm-
  • Medium (>=768px wide)
    Class prefix: ega-col-md-
  • Large (>=992px wide)
    Class prefix: ega-col-lg-
  • Extra large (>=1200px wide)
    Class prefix: ega-col-xl-

The classes above can be combined to create more dynamic and flexible layouts.
